# India Population Data Analysis 🇮🇳

End-to-end analysis of 2011 Indian Census data for 36 states/UTs using Python.

## 🚀 Run on Colab
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/aasthajain211/India-population-analysis/blob/main/analysis.ipynb)

## 📊 Key Findings
- **Most Urbanized:** Goa - 62.17%
- **Highest Literacy:** Kerala - 94%
- **Total Population:** 1.2 Billion

## 🛠️ Tools Used
Python, Pandas, Matplotlib, Google Colab
